EM ORDEM
	1-CRIAR IMAGEM
	2-CRIAR CONTAINER

1-CRIAR A IMAGEM NO DOCKER:
	docker-compose build --no-cache --force-rm

2-CRIAR O CONTAINER A PARTIR DA IMAGEM
	docker-compose up -d
3-REMOVER ".EXAMPLE" DO ARQUIVO ".ENV"

4-CRIAR A KEY DO PROJETO:
	docker exec laravel-electron bash -c "php artisan key:generate"

5-PARA O BANCO DE DADOS
	docker exec laravel-electron bash -c "php artisan migrate --seed"

6-LINK LARAVEL:
	http://localhost:9000/public
7-LINK PHPMYADMIN:
	http://localhost:9001
