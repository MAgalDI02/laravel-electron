FAZER OS COMANDOS EM ORDEM:

1-CRIAR A IMAGEM NO DOCKER E FAZER O CONTAINER A PARTIR DA IMAGEM:
	docker compose up -d --build

2-REMOVER ".EXAMPLE" DO ARQUIVO ".ENV"

3-ATUALIZAR O COMPOSER
	docker exec laravel-electron bash -c "composer update"

4-CRIAR A KEY DO PROJETO:
	docker exec laravel-electron bash -c "php artisan key:generate"

5-PARA O BANCO DE DADOS
	docker exec laravel-electron bash -c "php artisan migrate --seed"

6-LINK LARAVEL:
	http://localhost:9000/public
7-LINK PHPMYADMIN:
	http://localhost:9001
